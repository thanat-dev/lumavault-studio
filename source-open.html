เปิด Page Source

ทำตามขั้นตอน 1–4 — เมื่อวาง Source แล้ว ระบบจะส่งกลับ LumaVault ให้อัตโนมัติ

1
คัดลอก Source URL
2
เปิดแท็บใหม่ → วาง URL → Enter

กด Ctrl+T เปิดแท็บใหม่ใน Chrome แล้ววาง URL จากขั้นตอนที่ 1 ลงใน address bar แล้วกด Enter
(Chrome ไม่อนุญาตให้เว็บเปิด view-source: อัตโนมัติ จึงต้องทำเองใน address bar)

3
กด Ctrl+A แล้ว Ctrl+C

ในแท็บ view-source ที่เปิดได้ กด Ctrl+A เพื่อเลือกทั้งหมด แล้ว Ctrl+C เพื่อคัดลอก

4
กลับมาวาง Source ที่นี่ (Ctrl+V)

คลิกในกล่องด้านล่างแล้วกด Ctrl+V — ระบบจะส่งกลับ LumaVault ให้เองทันที

✓ ส่ง Source กลับ LumaVault แล้ว — ปิดแท็บนี้ได้เลย
